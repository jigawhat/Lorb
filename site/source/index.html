How to use The Lorb
  1. Queue up for a game
  2. (Optional) declare your champion
  3. If your team is on red side, click the "swap sides" button
  4. Copy and paste the full chat log into the grey box above
  5. Reorder the players by clicking & dragging them
  6. Select champion choices using the left-side buttons
  7. If an auto-assigned role is incorrect, select the correct one from dropdown menu
  8. (Optional) reconsider playing if the game is not in your favour ( use your judgement! )
